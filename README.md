## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
         npm install

- Run the development web server using the `node express_server.js` command.
- You can also run 'npm start' to start the app
